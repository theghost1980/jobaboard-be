Backend Server built on Heroku

Description: This is the backend service for jobaboard.net.

Details

This backend allow users to handle their accounts. The platform serves as a job posting platform but using NFTs. Each job is a NFT which users can collect, trade, buy or sell. Each user can have as many NFTs as they want and assign that NFT to a particular job, as an employee or as an employer.

This server was built using:

  • NodeJs & Express
  • Mongo Database
  • Cloudinary as image handler
  • HIVE blockchain technologies: NFTs & crypto loggin system.

How to use it

  • You must own a HIVE account to login into the platform.
  • Then configure your account using the Hive Keychain extension.
  • Now you can log in to Jobaboard.net
